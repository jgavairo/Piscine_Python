# Part.1 - Manipulation d'images et calculs

Cette partie couvre la manipulation d'images avec NumPy et PIL, ainsi que des calculs de base.

## Exercices

- **ex00** - `give_bmi.py` : Calcul de l'IMC (Indice de Masse Corporelle) à partir de listes de tailles et poids
- **ex01** - `array2D.py` : Découpage de tableaux 2D avec numpy
- **ex02** - `load_image.py` : Chargement et conversion d'images en tableaux NumPy
- **ex03** - `zoom.py` : Zoom sur une région spécifique d'une image
- **ex04** - `rotate.py` : Rotation d'images
- **ex05** - `pimp_image.py` : Application de filtres sur les images (niveaux de gris, etc.)
