# Part.2 - Data Science

Cette partie introduit l'analyse de données avec pandas et la visualisation avec matplotlib.

## Exercices

- **ex00** - `load_csv.py` : Chargement de fichiers CSV avec pandas
- **ex01** - `aff_life.py` : Visualisation de l'espérance de vie par pays
- **ex02** - `aff_pop.py` : Visualisation de l'évolution de la population (France et Suisse)
- **ex03** - `projection_life.py` : Analyse de corrélation entre espérance de vie et PIB par habitant
