# Part.3 - Programmation Orientée Objet

Cette partie explore les concepts avancés de la programmation orientée objet en Python.

## Exercices

- **ex00** - `S1E9.py` : Classes abstraites et héritage simple (Character, Stark)
- **ex01** - `S1E7.py` : Héritage et méthodes de classe (Baratheon, Lannister)
- **ex02** - `DiamondTrap.py` : Résolution du problème du diamant avec héritage multiple (King)
- **ex03** - `ft_calculator.py` : Surcharge d'opérateurs (+, -, *, /) pour une classe calculatrice
- **ex04** - `ft_calculator.py` : Opérations vectorielles avancées

## Prérequis

Aucune dépendance externe requise.
