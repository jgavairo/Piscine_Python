# Part.4 - Fonctions avancées

Cette partie couvre les fonctionnalités avancées de Python : décorateurs, closures et arguments variables.

## Exercices

- **ex00** - `statistics.py` : Fonction avec arguments variables (*args, **kwargs) pour calculer des statistiques
- **ex01** - `in_out.py` : Closures et fonctions d'ordre supérieur (outer, square, pow)
- **ex02** - `callLimit.py` : Décorateur pour limiter le nombre d'appels d'une fonction
- **ex03** - `new_student.py` : Classe avec génération automatique d'identifiants uniques
